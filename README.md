# Chat App

## Description
This Chat App is designed to facilitate real-time communication across multiple platforms. The project serves as an ideal demonstration of my skills in software development, particularly in building user-friendly interfaces and implementing effective back-end solutions.

## Features
- **Real-Time Messaging**: Users can send and receive messages instantly.
- **User Authentication**: Secure login and registration mechanisms.
- **Multi-Platform**: Accessible via web and mobile devices.
- **Group Chats**: Ability to create and join group conversations.
- **Message History**: Users can view past messages.

## Technology Stack
- **Frontend**: React.js
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **WebSocket**: For real-time communication

## Installation
To run the project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Rishiraj029/Chat-App.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Chat-App
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```

## Usage
1. Open the application in your web browser at `http://localhost:3000`.
2. Create an account or log in with existing credentials.
3. Start chatting!
